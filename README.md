# Notas

Options:
```
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base de la tabla de multiplicar
                                               [número] [requerido] [defecto: 0]
  -l, --list     Muestra la tabla de multiplicar por consola
                                                     [booleano] [defecto: false]
  -t, --to       Hasta qué número se multiplica la base   [número] [defecto: 10]
```
